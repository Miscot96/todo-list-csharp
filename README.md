# To-Do List (C#)

A console-based To-Do List application built with C# and .NET.

## Features
- Add tasks
- Edit tasks
- Complete tasks
- Remove tasks
- Sort by priority
- Save and load tasks (JSON)

## Project structure
- `Program.cs` - main program and user interaction
- `ToDoList.cs` - task management logic
- `ToDoTask.cs` - task model

## Technologies
- C#
- .NET
- System.Text.Json

## How to run
1. Open in Visual Studio
2. Run the project
